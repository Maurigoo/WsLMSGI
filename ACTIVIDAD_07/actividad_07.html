HTML 5

HTML5 es la última evolución del lenguaje HTML (HyperText Markup Language) y se ha convertido en el estándar principal para la construcción de páginas web modernas. Introducido a principios de la década de 2010, HTML5 trajo consigo una serie de nuevas características y mejoras que permitieron a los desarrolladores crear sitios web más interactivos, dinámicos y compatibles con una amplia variedad de dispositivos y navegadores.


    Algunas de las características o etiquetas introductorias son:

  • Section
  • Nav
  • Header
  • Footer

    Tambien tenemos mas etiquetas como por ejemplo:

  • Body
  • P
  • Img

CSS

CSS, que significa "Cascading Style Sheets" en inglés, es un lenguaje de hojas de estilo utilizado para controlar la presentación y el diseño de documentos HTML. En otras palabras, CSS se encarga de dar estilo y formato a elementos HTML, lo que permite crear páginas web atractivas y bien estructuradas.


    Los conceptos fundamentales de CSS son:

  • Selectores
  • Propiedades
  • Valores

    Tambien hay conceptos importantes como:

  • Cascada
  • Estilos de linea
  • Comentarios
  • Unidades de medida
